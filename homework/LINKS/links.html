THIRD EXAMPLE (Paragraphs)

Solution: каждую ссылку поместил в отдельный параграф, затем для параграфов задал свойство display: inline; с помощью селектора div.thirdExample>div.container>p, чтобы расположить параграфы в строку. Разделитель задан селектором div.thirdExample>div.container>p:nth-child(n+1):not(:last-child), свойство: border-right: solid 3px red;

FOURTH EXAMPLE (div)

Solution: применил блочную верстку. Она оказалась самой сложной для меня - не удалось добиться выравнивания навигейшен-меню по-центру экрана. Прошу Вашей помощи - помогите выровнять элементы по-центру экрана, пожалуйста. Вложил ссылки в отдельные div атрибуты, задал для div-ов свойство float: left;, чтобы разместить их в одну линию. Разделитель задан селектором div.fourthExample>div.container div[class*="Link"]:not(:last-child), свойство: border-right: solid 3px red;
